# February_Website

PS1-style FF7-inspired Three.js demo that loads Tifa from:

`/Users/yugrajdhillon/February_Website/public/assets/characters/tifa/Tifa/Tifa.dae`

## Run

1. Open a terminal in `/Users/yugrajdhillon/February_Website`
2. Start a static server:

```bash
python3 -m http.server 4173
```

3. Open `http://localhost:4173`

## Controls

- `W` / `S` or `Up` / `Down`: move
- `A` / `D` or `Left` / `Right`: turn
